[01] Approach

Most rooms are louder
than they are good.

We build sound systems the way a luthier builds an instrument. Slowly, by hand, with the room itself as part of the design. Every cabinet is engineered for the space it lives in. Every driver chosen for the music you actually listen to. The result is not "loud." The result is present.

From a two-channel listening room in a craftsman bungalow to a 40,000-watt rig for a festival main stage, the discipline is the same.

[04] Process

How a build happens.

  1. 01

    Conversation

    We talk about the room, the music, and how loud you actually want to live. No catalogue.

  2. 02

    Design

    Cabinets, drivers, crossover, amplification, room treatment. Proposed as one system, not a list of parts.

  3. 03

    Build

    Every cabinet is built on our floor. Birch ply, bracing, damping, finish, all done by hand and signed when it ships.

  4. 04

    Install & tune

    We install it, measure the room, and tune it in person. The day it's done is the day it sounds right.
